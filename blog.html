EXCITING NEW PROJECT

26/01/2018 // ADAM LEHODEY

This week, I've started the planning and initial design for a new project, all relating to mathematics! The project will involve solving and modelling difficult mathematical problems, including aspects from Analystics and Statistics using DataBases, Modelling with Quadratics, Expanding (a+bx)^y terms up to any point using binomial expansion, and using all other functions to complete things such as modelling using differentiation and finding the area under curves.

I was inspired to start this project through our Maths Key Assesment, which made me question in what ways I could incorporate both Maths and computing together to solve real world problems. One of the unique things about this project is that I am designing a fluent and beautiful graphical UI for users to interact with, creating a much better end project, including animations, through the use of JavaFX.

This week, I have created one of the base layers, the Binomial Expansion part of the project, which will work as a function, enabling other parts of the program such as harder modelling to work. By working this way, I am utilising some aspects of Functional Programming, using functions for abstraction. I have also worked on the user object and user login classes, which will allow users to keep notes and save their past answers to problems. In the following week, I will be working on a main class to start bringing together all these elements, and start work on the GUI, using MVC programming that I spoke about last week.
